# GastroFPS

Rankedowa gra multiplayer 5v5 o pracy w gastronomii, inspirowana Hells Kitchen i The Bear + kompetycja CS:GO. Unreal Engine 5.7.4, Windows, Steam.

## Quick start (Windows)

1. Zainstaluj Epic Games Launcher → Unreal Engine 5.7.4 (~50GB)
2. Zainstaluj Visual Studio 2022 Community + workload "Game development with C++"
3. Zainstaluj Git + Git LFS
4. Sklonuj to repo
5. Otwórz `GastroFPS/GastroFPS.uproject` → build (VS first-time compile ~5min)
6. Play w Editor
